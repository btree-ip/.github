## 회사 소개

**비트리(BTree)**는 2014년 설립된 반도체 설계자산(IP) 전문 기업으로, 경기도 성남시 분당구에 본사를 두고 있습니다. 고성능·저전력 IP 코어를 설계하며, **이미지 캡처(ISP), 비전 처리(Vision), 영상 압축(Codec), 디스플레이 출력(Display)**에 이르는 비디오 신호 처리 전 과정을 아우릅니다.

단일 스테이지에 특화된 IP 벤더와 달리, 비트리는 **비디오 파이프라인 전체에 걸친 시너지**를 제공하여 고객이 일관된 화질, 최적화된 레이턴시, 축소된 실리콘 면적을 갖춘 SoC를 구현할 수 있도록 지원합니다.

## 핵심 기술

```
┌─────────────────────────────────────────────────────────────────┐
│                  End-to-End 비디오 파이프라인                      │
│                                                                 │
│   Camera       Vision/AI      Codec          Display            │
│  ┌───────┐    ┌───────┐    ┌───────┐    ┌───────────┐          │
│  │  ISP  │───▶│ Vision│───▶│ Codec │───▶│ Display   │          │
│  │  IP   │    │  IP   │    │  IP   │    │ Driver IP │          │
│  └───────┘    └───────┘    └───────┘    └───────────┘          │
│                                                                 │
│  Bayer RAW     객체 인식      인코딩/      DDI + 패널             │
│  → YUV/RGB     ADAS, AI     디코딩       압축 전송               │
│  HDR, 3DNR                   VDC-M        VESA DSC              │
└─────────────────────────────────────────────────────────────────┘
```

### 1. ISP (Image Signal Processor)

비트리의 플래그십 **METASEQUOIA** ISP IP는 비디오 파이프라인의 첫 번째 단계를 담당하며, 이미지 센서의 RAW 데이터를 고품질 영상으로 변환합니다.

| 항목 | 사양 |
|---|---|
| 해상도 | 최대 **13MP @ 60fps** (단일 인스턴스) |
| 멀티 인스턴스 | 최대 **4개 동시 입력** |
| Bayer 입력 | 8 / 10 / 12 / 14-bit RAW |
| 출력 포맷 | YUV420, YUV422, YUV444, RGB888 |
| 인터페이스 | DVP, AXI |
| 주요 기능 | HDR, 3D 노이즈 리덕션(3DNR), 렌즈 왜곡 보정(LDC), 디워핑, 스케일링 |

### 2. Vision & AI 처리

비트리의 비전 IP는 이미지 캡처와 지능형 장면 인식 사이의 간극을 연결합니다.

- **자동차** — AVM(Around View Monitoring), SVM(Surround View), ADAS, 블랙박스
- **보안/감시** — 실시간 모니터링 및 영상 분석
- **Edge AI** — ISP 단의 최적화된 전처리를 활용한 온디바이스 추론

ISP와 함께 비전 알고리즘을 **공동 설계(co-design)**함으로써, 캡처와 처리 사이에서 불필요한 포맷 변환이나 대역폭 낭비 없이 **무손실 핸드오프**를 실현합니다.

### 3. 비디오 코덱 (Video Codec)

ISP에서 생성된 고해상도 스트림을 효율적으로 전송·저장하기 위해 압축은 필수입니다. 비트리의 코덱 IP 포트폴리오:

- **VDC-M 1.2 인코더** — VESA Display Compression-M 준수, 최대 **5:1 압축비(6bpp)**에서 시각적 무손실
- **VDC-M 1.2 디코더** — 파이프라인 전체의 무결성을 보장하는 매칭 디코더
- **커스텀 압축** — 대역폭 제약 환경을 위한 애플리케이션별 코덱 최적화

코덱 IP는 상류의 ISP, 하류의 디스플레이 단과 **동기화(lockstep)** 동작하도록 설계되어, 레이턴시를 최소화하면서 시각적 충실도를 유지합니다.

### 4. 디스플레이 (DDI)

파이프라인의 마지막 단계 — 비트리의 **DDI(Display Driver IC) IP**는 처리된 프레임을 패널에 정확하게 전달합니다.

- 모바일 및 자동차 패널에 최적화된 Display Driver Interface IP
- 디스플레이 링크의 대역폭 절감을 위한 **VESA 디스플레이 압축** 통합 지원
- 코덱 단과 공동 최적화하여 디코드-재인코드 오버헤드 제거

## 파이프라인 시너지의 차별점

대부분의 IP 벤더는 개별 솔루션(point solution)을 제공합니다. 비트리의 차별화는 **스테이지 간 교차 최적화**에 있습니다.

| 과제 | 개별 솔루션 방식 | 비트리 파이프라인 방식 |
|---|---|---|
| ISP→Vision 핸드오프 | 포맷 변환 오버헤드 발생 | 네이티브 포맷 패스스루, 공동 설계 인터페이스 |
| 압축 아티팩트 | 자연 이미지용 범용 코덱 | ISP 출력 특성을 인지하는 코덱 |
| 디스플레이 대역폭 | 디스플레이 링크에서 별도 압축 | 캡처부터 패널까지 통합 압축 전략 |
| 멀티 카메라 시스템 | 카메라별 독립 처리 | 멀티 인스턴스 지원 공유 ISP + Vision 파이프라인 |
| 총 실리콘 면적 | 개별 IP 블록의 단순 합산 | 공유 버퍼, 통합 제어, 게이트 수 절감 |

하나의 팀이 **ISP, Vision, Codec, Display** 전체 체인을 설계하면, 모든 인터페이스가 호환성 문제가 아닌 **최적화의 기회**가 됩니다.

## 주요 연혁

- **2014** — 경기도 성남시에서 창립
- **2016** — SK텔레콤, ID Quantique와 QRNG 칩셋 공동 개발 시작
- **2020** — 세계 최초 **양자난수생성기(QRNG)** 칩 상용화 (삼성 갤럭시 A 퀀텀 탑재)
- **2024** — 창립 10주년; IP-SoC Korea 참가; 코스닥 상장 준비

## 비즈니스 모델

비트리는 고객의 공정별 실리콘 재검증이 불필요한 **소프트웨어 기반 IP**를 제공합니다.

- **낮은 통합 비용** — 팹리스 및 디자인하우스 고객의 도입 비용 절감
- **공정 이식성** — 파운드리 노드에 독립적
- **지속적 로열티 수익** — 고객 출하량에 연동된 수익 구조
